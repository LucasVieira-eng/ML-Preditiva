# 📊 ML Preditiva

Projeto de **Machine Learning Preditivo** desenvolvido com o objetivo de explorar técnicas de **análise de dados e modelagem preditiva**, utilizando Python e bibliotecas de ciência de dados.

Este repositório apresenta o processo completo de desenvolvimento de um modelo de aprendizado de máquina, incluindo **análise exploratória de dados, pré-processamento, treinamento de modelos e avaliação de desempenho**.

---

# 🚀 Objetivo

O objetivo deste projeto é aplicar conceitos de **Machine Learning** para construir modelos capazes de **identificar padrões nos dados e realizar previsões**.

Durante o desenvolvimento foram exploradas etapas importantes de um projeto de ciência de dados, como:

- Análise exploratória dos dados (EDA)
- Limpeza e preparação dos dados
- Engenharia de atributos (Feature Engineering)
- Treinamento de modelos de Machine Learning
- Avaliação de desempenho do modelo

Projetos de ML geralmente seguem esse fluxo para transformar dados brutos em modelos capazes de realizar previsões ou classificações.

---

# 🧠 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes ferramentas e bibliotecas:

- **Python**
- **Pandas** – Manipulação e análise de dados
- **NumPy** – Computação numérica
- **Matplotlib** – Visualização de dados
- **Tensorflow** – Modelagem de Machine Learning
- **Streamlit**

---

# 📊 Etapas do Projeto

## 1️⃣ Análise Exploratória de Dados (EDA)

Nesta etapa os dados foram analisados para compreender:

- Distribuição das variáveis
- Correlações entre atributos
- Identificação de valores ausentes
- Possíveis padrões nos dados

---

## 2️⃣ Pré-processamento dos Dados

Foram realizadas etapas de preparação dos dados para melhorar o desempenho dos modelos:

- Tratamento de valores ausentes
- Normalização ou padronização
- Codificação de variáveis categóricas
- Seleção de variáveis relevantes

---

## 3️⃣ Treinamento do Modelo

Os modelos de Machine Learning foram treinados utilizando bibliotecas do ecossistema Python.

Nesse projeto foi utilizado Tensorflow e tecnologia Convolutional Neural Network (CNN).

---

## 4️⃣ Avaliação do Modelo

Após o treinamento, o modelo foi avaliado utilizando métricas como:

- Mean Squared Error (MSE)
- Mean Absolute Error (MAE)

---

# 📈 Resultados

Os resultados obtidos demonstram como técnicas de **Machine Learning podem ser utilizadas para extrair conhecimento dos dados e gerar previsões úteis**.

O projeto também demonstra a importância de:

- Bons dados
- Pré-processamento adequado
- Escolha correta do modelo
